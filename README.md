# Charlie Qi Portfolio Website

Personal one-page portfolio for graduate and junior software engineering, Python backend, data, automation and AI-adjacent roles.

## Tech Stack

- React
- Vite
- CSS
- lucide-react

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Build

```bash
npm run build
```

## Notes

The resume PDF is stored in `public/charlie-qi-resume.pdf` for the resume button.
